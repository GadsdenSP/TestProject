Project Anne – Protocol Flow

1. Create Content

The user writes, records, or generates something — text, image, audio, or video.

↓

2. Generate Hash

The content is hashed using SHA-256. This creates a unique fingerprint.

↓

3. Add Metadata

Attach origin info: creator ID, device data, AI model (if applicable), and timestamp.

↓

4. Sign Content

The creator digitally signs the hash and metadata with their private key or wallet.

↓

5. Build Anne Package

Everything is wrapped into a verifiable JSON object — the Anne Package.

↓

6. Add Endorsements (Optional)

Trusted parties can review and endorse the package, adding signatures and comments.

↓

7. Verify Integrity

Anyone can later check the hash, metadata, and signatures to verify authenticity.
