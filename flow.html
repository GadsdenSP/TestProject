Project Anne – Protocol Flow

1. Create Content

The user writes, records, or generates something — text, image, audio, or video.

↓

2. Generate Hash

The content is hashed using SHA-256. This creates a unique fingerprint.

↓

3. Add Metadata

Attach origin info: creator ID, device data, AI model (if applicable), and timestamp.

↓

4. Sign Content & Metadata

The creator digitally signs the hash and metadata with their private key or wallet. This binds them to the content and context.

↓

5. Build Anne Package

Everything is wrapped into a verifiable JSON object — the Anne Package.

↓

6. Validation & Trust Layer (Optional)

Validators (humans, DAOs, AI agents) review the Anne Package and may endorse it by signing. These signatures increase trust and help platforms make informed decisions.

↓

7. Verification

Any verifier can now check:

  • ✔️ Content hash matches the actual file
  • ✔️ Origin signature is valid
  • ✔️ Metadata has integrity
  • ✔️ Endorsements are real and optionally trusted

This process reveals if content is authentic, manipulated, or unverifiable.
